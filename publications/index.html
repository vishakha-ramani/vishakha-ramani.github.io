  1. Performance Modeling for Disaggregated Prefill-Decode LLM Inference Serving
    Vishakha Ramani and Asser Tantawi
    2026
    Abs

    Modern large language model inference systems increasingly disaggregate the prefill and decode phases of generation onto distinct hardware pools. This separation improves utilization, but it also breaks the single-queue abstraction that classical queueing analyses rely on. We develop a performance model for disaggregated prefill-decode (P/D) inference that treats the two phases as coupled queueing stations with phase-dependent service requirements. The model captures the interaction between prefill batching, decode iteration scheduling, and KV-cache transfer, and yields closed-form expressions for end-to-end latency and throughput under realistic request mixes. We use the model to characterize when disaggregation strictly dominates a co-located baseline and to derive sizing rules for the prefill and decode pools.

  2. A Queueing-Theoretic Framework for Scheduling and Autoscaling LLM Inference
    Vishakha Ramani
    2026
    Abs

    Production large language model inference deployments must hold latency Service Level Objectives under bursty, heterogeneous request mixes while keeping accelerator utilization high. We propose a queueing-theoretic framework that unifies request-level scheduling and replica-level autoscaling for prefill-decode disaggregated systems. The framework treats Service Level Objective compliance as a stability constraint on a multi-class queueing network and uses Lyapunov drift-plus-penalty optimization to derive scheduling and scaling policies that are provably stable whenever the offered load is within capacity. We show how the framework specializes to the Workload Variant Autoscaler design used in the open-source llm-d project and how it generalizes to alternative SLO formulations.
